# ペットなまえ診断 — 設計ドキュメント

**作成日**: 2026-04-03  
**フェーズ**: MVP (Phase 1)

---

## 概要

ペットの新規オーナー向け、条件フィルターからペットの名前候補を提案するWebサービス。  
将来的には写真からAIが雰囲気を読み取り名前を提案する機能（Phase 2）を追加予定。

---

## ターゲットユーザー

- 犬・猫・うさぎ・ハムスター・鳥などを新しく迎えたオーナー
- スマートフォン・PCどちらからもアクセス

---

## ビジネスモデル

- **Phase 1**: 広告スペースを確保するが未掲載（将来のAdSense設置に備えたレイアウト）
- **Phase 2**: Google AdSense導入（サイドバー300×250、インフィードバナー）
- **バイラル**: SNSシェア機能で自然流入を促進

---

## 機能要件（MVP）

### フィルター（4種類）

| フィルター | 選択肢 |
|---|---|
| 動物の種類 | 犬 / 猫 / うさぎ / ハムスター / 鳥 |
| 性別 | オス / メス / どちらでも |
| 雰囲気 | かわいい / かっこいい / 個性的 / 和風 / 洋風 / ふわふわ系 |
| 毛色 | 白 / 黒 / 茶色 / 三毛 |

- 複数選択可（OR条件でマッチング）
- フィルター変更時にリアルタイムで結果更新

### 名前候補表示

- カード形式（名前・読み・タグ表示）
- マッチ件数表示
- 人気順・ランダム並び替え

### 名前データベース

- JSONファイルで管理（約300件）
- 各名前に以下のタグを付与：`species[]`, `gender`, `vibe[]`, `color[]`, `reading`, `meaning`

### シェア機能

- 「この名前をシェア」ボタン → X（Twitter）シェア
- OGP設定でシェア時に名前カードが表示される

### レイアウト

- 2カラム（メインコンテンツ + サイドバー）
- サイドバーには広告スペース（空白）と人気ランキングウィジェット
- モバイルでは1カラムにフォールバック

## 技術スタック

| 要素 | 選択 |
|---|---|
| フロントエンド | HTML + CSS + Vanilla JS（フレームワークなし） |
| データ | names.json（ローカル、バックエンド不要） |
| ホスティング | Vercel または GitHub Pages（無料） |
| OGP | 静的metaタグ |
